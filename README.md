# Fluency Link Academy

## Описание проекта

Это демонстративный веб-сайт для школы английского языка "Fluency Link Academy", ориентированный на индийский рынок. Проект представляет собой статический сайт с информацией о курсах, услугах и контактах академии.

## Страницы сайта

- **Главная (index.html)**: Введение в академию и основные преимущества.
- **Услуги (service.html)**: Описание предлагаемых курсов английского языка, таких как General English, Business English, IELTS/TOEFL Preparation и другие.
- **О нас (about.html)**: Информация о команде и миссии академии.
- **Контакты (contact.html)**: Форма обратной связи и контактные данные.
- **Дополнительные страницы**: Политика конфиденциальности, условия использования, благодарность за заявку.

## Технологии

- **HTML**: Структура страниц.
- **CSS**: Стилизация (файл style.css).
- **JavaScript**: Интерактивные элементы (файл script.js).
- **Изображения и иконки**: В папках images/ и icons/.

## Как запустить

1. Скачайте или клонируйте репозиторий.
2. Откройте файл `index.html` в любом современном веб-браузере.
3. Сайт готов к просмотру без необходимости сервера.

## Цель проекта

Проект создан для демонстрации возможностей академии по обучению английскому языку, с акцентом на индийский рынок. Он включает адаптированные материалы и контактную информацию для региона.
